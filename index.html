🎮 Changer son pseudo Minecraft Java

Guide rapide pour modifier gratuitement votre pseudo Minecraft Java Edition.

📖 Étapes

1

Aller sur Minecraft.net

Ouvrez le site officiel Minecraft.

2

Connexion Microsoft

Connectez-vous à votre compte Microsoft.

3

Profil Java Edition

Accédez à Profil → Minecraft Java Edition.

4

Modifier le pseudo

Cliquez sur « Change Profile Name ».

5

Valider

Choisissez votre nouveau pseudo et confirmez.

🌍 Ouvrir Minecraft.net

ℹ️ Informations importantes

💸 Gratuit

Le changement de pseudo est entièrement gratuit.

⏳ 30 jours

Un changement de pseudo est autorisé tous les 30 jours.

🎮 UUID

Vos mondes et achats restent liés à votre compte.
