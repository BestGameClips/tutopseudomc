🎮 Changer son pseudo Minecraft Java

Guide complet pour modifier gratuitement votre pseudo Minecraft Java Edition.

📖 Étapes à suivre

1

Aller sur Minecraft.net

Ouvrez le site officiel Minecraft.

2

Connexion Microsoft

Connectez-vous avec votre compte Microsoft.

3

Ouvrir le profil

Accédez à Profil → Minecraft Java Edition.

4

Changer le pseudo

Cliquez sur « Change Profile Name ».

5

Valider

Entrez votre nouveau pseudo puis confirmez.

ℹ️ À savoir

💸 Gratuit

Le changement de pseudo est totalement gratuit.

⏳ 30 jours

Vous ne pouvez modifier votre pseudo qu'une fois tous les 30 jours.

🎮 UUID

Vos mondes et achats restent liés à votre compte et non au pseudo.
